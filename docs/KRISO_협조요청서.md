안녕하십니까.

「대화형 해사서비스 플랫폼을 위한 지식그래프 모델 설계 연구 용역」을 수행하는 **㈜인사이트마이닝**입니다.

본 용역은 계약 후 240일(8개월)의 단기 설계 연구로, 주요 산출물의 마감이 M+2·M+5·M+7·M+8 시점에 집중되어 있습니다. 특히 연구소 보유 자료·데이터·환경 등 연구소 제공이 전제된 항목이 다수 있어, 원활한 수행을 위하여 아래 사항의 협조를 요청드립니다.

## 1. 자료·데이터 제공 요청

| # | 협조 사항 | 요청 시기 | 연계 |
|---|-----------|----------|------|
| A-1 | 시험시설 8종 데이터 (메타데이터 목록 → 명세 문서 → 실측 데이터, **단계적 제공** — 아래 ※ 참조) | 1차 착수 후 2주 / 2·3차 후속 | REQ-001 |

| A-3 | 내부 실험관리시스템(LIMS) 스키마 문서 또는 접근 경로 — 보안등급 지정 시 테이블명·필드명 수준 부분 공개 또는 연구소 내 열람으로 대체 가능, 미제공 시 ScholarWorks@KRISO 공개 메타데이터로 우선 대체 | 착수 후 1개월 | REQ-001·DES-003 |

| A-6 | 연구소 보안 지침·규정 문서 (해사 데이터 보안 지침) | 착수 시 | COM-001 |
| A-7 | 데이터 보안등급 분류 정보 (보안등급 지정 데이터 식별) | A-1과 함께 | COM-001·REQ-001 |
| A-8 | PoC 검증용 샘플 데이터 3종 지정·제공 | M+3 | TER-001 |
| A-9 | 연구소 보유 검증용 소프트웨어 제공 (보유 시) | M+5 이전 | COM-004 |

> **※ 시험시설 데이터(A-1) 단계적 제공 요청**
>
> 본 용역에 최종적으로 필요한 시험시설 데이터의 **전체 범위**는 다음 3가지입니다.
> - ① **메타데이터 목록** — 시설별 생성 데이터 종류·형식·갱신주기 등 ([첨부 1] 양식)
> - ② **데이터 명세·설명 문서** — 데이터 사전, 시험조건 명세서, 계측채널 정의서 등
> - ③ **실측 샘플 데이터** — 원천·전처리·분석 결과 데이터
>
> 데이터 본체(②③)는 준비에 시간이 소요될 수 있으므로, **1차로 메타데이터 목록([첨부 1] 양식)을 착수 후 2주 내 우선 제공**해 주시고, 명세 문서(②)는 착수 후 1개월, 실측 샘플 데이터(③)는 M+3 전후로 후속 제공해 주시기를 요청드립니다.
>
> ※ **메타데이터 목록만으로는 산출물의 '틀'까지만 작성 가능**하며, 실제 시험시설 자원구조 분석을 위해서는 ②③의 제공이 **반드시 필요**합니다. ②③의 준비 일정을 1차 회신 시 함께 알려주시면 전체 일정 조율에 반영하겠습니다.

## 4. 당사 자체 조사 범위 및 추가 참고자료 요청

본 용역의 다음 항목은 당사가 **공개 자료를 기반으로 자체 조사**하여 수행하며, 별도 협조 없이 진행합니다.

- 그래프 DB 기술 동향 (LPG/RDF, Neo4j 등) — REQ-003
- 해사 분야 국제 표준 (IHO S-100 시리즈, IMO GMDSS, AIS 등) — REQ-004
- 해사 도메인 지식그래프 학술 선행연구 (MAKG, PIKG, 오재용·김혜진 2024 등) — REQ-004·DES-001
- 외부 기관 공개 데이터 현황 (공개 API·데이터 포털)

다만 위 조사의 완결성과 본 용역의 품질을 높이기 위하여, 연구소가 보유·인지하고 계신 다음 정보가 있으시면 알려주시면 **적극 참고·반영**하겠습니다.

- 해사 분야 참고 문헌·표준·웹 자료(URL)
- 연구소 선행 연구·기획 자료
- 분야별 내부 전문가 또는 자문 가능 인력
- 기타 본 용역 수행에 도움이 될 자료·정보

구체적인 협조 시기는 착수보고 회의에서 협의·확정하고자 합니다.
