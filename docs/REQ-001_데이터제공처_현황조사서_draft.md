# 해사서비스 플랫폼 데이터 제공처별 현황 조사서 (Draft)

## REQ-001 산출물 ① · 데이터 제공처별 현황

> **Draft v0.2 — 2026-05-20**
> 외부 기관 = 공공데이터포털(data.go.kr) 및 기관 공식 사이트 **실조사** — 60개 이상 데이터셋 확인.
> 내부 시험시설 = KRISO 협조 자료(협조요청 A-1) 수령 후 작성.
> 근거: 과업지시서 REQ-001, 기술제안서 II.2

## 1. 개요

### 1.1 목적
지식그래프로 구축 가능한 멀티모달 해사데이터의 제공처별 현황을 조사하고, 데이터 간 연계 가능성과 활용도를 평가한다.

### 1.2 조사 결과 요약
| 제공처 | 확인 데이터셋 | 주력 형식 | 핵심 식별자 |
|--------|--------------|----------|------------|
| 해양수산부 (GICOMS·Port-MIS) | 27건 | CSV·OpenAPI·GIS | MMSI·IMO·호출부호·UN/LOCODE |
| 국립해양조사원 (KHOA) | 15건 (+API 240여 개) | S-57·JSON API·SHP | ENC셀번호·ObsCode |
| 국립해양측위정보원 (NMPNT) | 5건 | RTCM·RINEX·JSON | 기준국ID·관측소코드 |
| 기상청 (KMA) | 7건 | CSV·JSON API | 지점번호·특보구역코드 |
| 해양경찰청 (KCG) | 6건 | CSV·REST API | 연번·일자+지역 |
| 자율운항선박실증센터 (KRISO) | 4건 | (대부분 비공개) | — |
| 기타 (MEIS·KIOST·중앙해양안전심판원) | 13건+ | CSV·API·NetCDF4 | 정점코드·사건번호 |

### 1.3 조사 항목 (11개)
데이터명 · 제공기관 · 생성목적 · 형식 · 갱신주기 · 시공간범위 · 주요필드 · 메타데이터 · 식별자 · 연계항목 · 접근권한

### 1.4 조사 방법
공공데이터포털·기관 공식 포털 실조사. 데이터셋별 메타·API 명세 확인. 출처 URL 기록.

## 10. 종합 — 데이터셋 간 연계 (KG 설계 핵심)

### 10.1 조인 키 매핑

| 조인 키 | 연결 데이터셋 | 표준 |
|---------|-------------|------|
| **IMO 번호** | 선박원부 ↔ 국제선박등록 ↔ 선박검사 ↔ AIS정보 ↔ 중앙해양안전심판원 사고 | IMO 국제 |
| **MMSI** | AIS 동적 ↔ AIS 정적 ↔ 연안AIS 통계 ↔ 인천항 AIS | ITU-R M.585 |
| **호출부호(Call Sign)** | 선박입항신고 ↔ 선박관제(VTS) ↔ 선박운항정보 ↔ 한중일 입출항 | ITU |
| **UN/LOCODE** | 국가항구코드 ↔ 차항지·목적항 필드 | UNECE |
| **ObsCode(관측소코드)** | 조석예보 ↔ 조위실측 ↔ 파고 ↔ 조류 ↔ 해양관측소 | KHOA |
| **정점코드** | 해양환경측정망 수질 ↔ CTD ↔ 퇴적물 | MEIS |
| **위경도(WGS84)** | 해양오염사고 ↔ 해양사고 통계 ↔ GOCI-II 격자 ↔ 조석·파고 관측소 | 공간 조인 |
| **ENC 셀번호** | 전자해도 ↔ 수심정보 ↔ 해안선 | IHO S-57 |
| **UN번호** | 위험물정보(IMDG) ↔ 입출항 위험물톤수 | UN IMDG |

### 10.2 접근성 매트릭스

| 접근 유형 | 데이터 예 |
|-----------|----------|
| 완전 공개 (로그인 불필요) | AIS 통계·항만정보·해안선·해양사고 통계 등 CSV 파일 다수 |
| API Key (자동승인) | 위험물정보·수심정보·해양환경측정망 |
| API Key (심의/바다누리) | 조석·조위·파고·조류 |
| 유료 구매 | 전자해도 ENC (S-57/S-63 원본) |
| 비공개 | LRIT·VMS·V-PASS 실시간 위치, 자율운항 실증 데이터, 해경 SAR 작전기록 |

### 10.3 핵심 발견
- **AIS·Port-MIS·선박원부**가 IMO·MMSI·호출부호로 강하게 연결 — 선박 중심 지식그래프의 핵심 축
- **조석·파고·조류·해양기상**이 ObsCode·위경도로 연계 — 해역·환경 노드
- 전자해도(ENC)는 유료이나 S-100 표준 매핑(REQ-004)의 핵심 — 활용 조건 협의 필요
- 자율운항 실증 데이터는 발주처(KRISO) 내부 자산 — 협조 필수
