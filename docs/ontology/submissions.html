제출 절차

COM-005 연계

GitHub 등록

모든 산출물을 감독관 GitHub 리포지토리에 유형별 폴더(01_data_analysis ~ 05_reports)로 등록·버전 관리.

PR 리뷰·승인

주요 산출물 제출 전 Pull Request로 감독관 리뷰를 받고 승인 후 병합.

감독관 확인·서명

성과물은 연구소 검사를 받으며, 감독관 확인·날인 후 검수 확정 → 기성·잔금 청구 가능.

전자문서 제출

과업지시서 II.9에 따라 모든 과업성과물을 전자문서 형태로 연구소에 제출.

⚠
보안 등급이 지정된 해사 데이터는 GitHub 등 외부 저장소 업로드 금지(COM-001). 산출물 문서만 GitHub로 관리하고, 원천 데이터는 연구소 내부 환경에 보관한다.
