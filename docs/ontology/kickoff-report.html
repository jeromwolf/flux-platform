용역 실시 계획서

과업 목적 — 해사 분야 멀티모달 데이터(해양관측·선박정보·전자해도·기상·항해 등)에 대한 지식그래프 모델을 설계하고, 데이터 간 의미적 관계를 체계화하여 자연어 기반 데이터 탐색 및 AI 서비스의 기반을 구축한다.

과업 범위 — 대화형 해사서비스 플랫폼의 지식그래프 모델을 설계하며, 2차년도 그래프 DB 구축·자연어 탐색 기능 개발로 이어지는 확장성 있는 설계를 수행한다. 5개년 2단계 사업의 1단계 1차년도(운영 개념 정의 및 설계)에 해당한다.

수행 체계 — 3단계 접근

단계명칭주요 내용
1단계해사 데이터 분석 및 구축 대상 선정외부·내부 데이터 현황 조사, 멀티모달 분류, 구축 대상·우선순위 도출
2단계해사 도메인 온톨로지 및 KG 모델 설계온톨로지 설계(스탠포드 7단계), 국제표준 정합성, 자원 관계 모델, 그래프 DB 비교
3단계지식그래프 모델 검증 PoC샘플 데이터 검증, 프로토타입 구축, 자연어 탐색 검토

요구사항별 수행 계획 — 15개

용역수행자 명단 및 편성표

해사 도메인 온톨로지 설계·그래프 DB 구현·PoC 개발 3개 전문 분야를 균형 있게 수행하기 위해 정규 인력 4명으로 구성한다. 과업지시서 보안 요건에 따라 외부 인력 참여를 제한한다.

역할성명소속·직급담당 요구사항
PM · 온톨로지 설계 책임양선희인공지능분석실 실장DES-001 · REQ-004 · DES-004 / 과제 총괄
그래프 DB 설계신동욱인공지능분석실 팀장REQ-003 · DES-002 · TER-001 · TER-002
데이터 품질이부일대표이사REQ-001 · REQ-002 · DES-003 (품질 총괄)
데이터 분석 · ETL이재현빅데이터분석실 팀장REQ-001 · REQ-002 · DES-003

※ 데이터 분석·ETL 담당은 이재현으로 확정. 정성제안서 표기('이동훈')는 착수보고 시 정정 보고한다(C-3). 투입 인월(M/M)은 편성표 확정 시 기재. 참여자 교체 시 연구소 사전 승인 후 보안각서 재제출(COM-001).
